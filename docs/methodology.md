# 방법론 노트

## 0. 기준 문서

- `AGENTS.md`: 작업 기준과 품질 기준
- `SOURCE.md`: 참고 경로와 컨텐츠 재구성 기준

## 1. 프로젝트 목적

저베타 자산을 레버리지하고 고베타 자산을 축소하는 BAB 아이디어를 단순 수치로 검증한다.

## 2. 핵심 질문

저베타 그룹을 시장 베타 1로 스케일하면 고베타 그룹 대비 기대 초과수익 스프레드가 남는가?

## 3. 입력 데이터

- 파일: `data/sample/asset_beta_panel.csv`
- 성격: 구조 검증용 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 베타 중앙값을 기준으로 저베타/고베타 leg를 나누고 각 leg를 베타 1로 스케일한다.
2. 스케일된 기대 초과수익 차이를 BAB spread의 첫 진단 지표로 둔다.
3. 수익률과 베타는 BAB leg 구성을 설명하는 교육용 합성 패널이다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
